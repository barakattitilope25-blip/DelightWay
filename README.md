# Hajih-Web Project

This project contains a React frontend and a Node.js/Express backend.

## Structure

- `frontend/`: React application (Vite-based)
- `backend/`: Node.js/Express server

## Getting Started

### Prerequisites

- Node.js and npm installed on your machine.

### Setup

1. **Frontend**:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

2. **Backend**:
   ```bash
   cd backend
   npm install
   npm run dev
   ```

## Technologies

- **Frontend**: React, Vite, ESLint
- **Backend**: Node.js, Express, Cors, Dotenv
